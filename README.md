# HTML5 Semantic Structure & Accessibility

A beginner-friendly project demonstrating the use of HTML5 semantic elements to create well-structured, accessible, and SEO-friendly web pages.

## 📌 Project Overview

This project showcases how to use modern HTML5 semantic tags to improve:

- Website structure
- Accessibility
- Search Engine Optimization (SEO)
- Code readability and maintainability

## 🏗️ Semantic Elements Used

| Element | Purpose |
|----------|---------|
| `<header>` | Website or section header |
| `<nav>` | Navigation links |
| `<main>` | Main page content |
| `<section>` | Related content grouping |
| `<article>` | Independent content |
| `<aside>` | Sidebar or supplementary content |
| `<footer>` | Footer information |

These semantic elements improve accessibility and help search engines understand content hierarchy. :contentReference[oaicite:1]{index=1}
